website

This website is a static HTML, CSS, and JS website built with Eleventy (11ty). I programmed, wrote, and deployed it myself. I think programming is a fun skill that anyone can learn. It has never been easier to build an 'old school' style website.

The font is JetBrains Mono. It's a font I love because it's used in IntelliJ, the Java IDE, and also because its free and open source.

The contact form is a free form built with staticforms.xyz. My domain is registered via NameCheap and hosted on Cloudflare as a free static website.

My mailing list is run with Mailerlite. I don't use a CMS like Wordpress or similar, I write blog posts in Markdown and then copy them into Mailerlite.
